
Hospital Inventory Management System

A simple web-based Hospital Inventory Management System designed to help hospitals track, manage, and monitor medical equipment efficiently. The system reduces manual paperwork, improves equipment tracking, and supports maintenance management.

Features
User Authentication (Login System)
Add/Register Medical Equipment
Update Equipment Information
Delete Equipment Records
Search Equipment Quickly
Filter Equipment by Different Categories
Record Maintenance History
Generate Equipment Reports
Dashboard for Inventory Overview
System Actors
Admin
Biomedical Staff
Hospital Staff
